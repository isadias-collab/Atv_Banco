# 🏦 Sistema Bancário - Atv_Banco

Projeto desenvolvido como atividade prática com foco na simulação de um sistema bancário, aplicando conceitos fundamentais de programação, validação de dados e lógica de negócios.

---

## 📌 Descrição

O **Atv_Banco** é um sistema que simula operações básicas de um banco, permitindo interação do usuário com funcionalidades como autenticação, controle de acesso e possíveis operações financeiras.

O projeto tem como principal objetivo reforçar a importância da **validação de dados** e da **segurança em sistemas bancários**, evitando erros como inserção de valores inválidos ou acessos indevidos.

---

## 🎯 Objetivos

- Implementar um sistema bancário simples
- Garantir validação correta de entradas de dados
- Aplicar lógica de autenticação de usuários
- Simular regras reais de segurança digital
- Praticar desenvolvimento web com HTML, CSS e JavaScript

---

## ⚙️ Funcionalidades

- 🔐 Sistema de login com validação de usuário e senha  
- 🚫 Controle de tentativas de acesso  
- 📊 Exibição de mensagens de status ao usuário  
- 🧠 Validação de dados inseridos (evitando erros críticos)  
- 🖥️ Interface simples e funcional
